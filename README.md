# ML_trading_project

ML Project for CS class at Ensae Paris. Topic : Implement a day-to-day trading strategy using tools from ML. Bagging, Boosting and Stacking techniques are used with an educational approach.

Are used : yfinance, finta, pandas, xgboost, sklearn.
